The Google Interactive Media Ads (IMA) SDK enables publishers to display linear, non-linear, and companion ads in interactive media content such as videos and games. Publishers can use the IMA SDK to request and display ads served by DoubleClick for Publishers (DFP), the Google AdSense network, or any VAST-compliant ad server. Common scenarios for using the IMA SDK include displaying pre-rolls or overlays in video content. Learn More.
